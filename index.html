eBayUI Opt-In Switches

Use the switches to opt-in or opt-out.

Load Switches

Select a file with the following format.

                      {
                        "switches" : [
                        {
                          "name" : "Some_Name",
                          "optInUrl" : "url",
                          "optOutUrl" : "url",
                          "defaultState" : "optin"
                        },
                        {
                          "name" : "Some_Name_2",
                          "optInUrl" : "url2",
                          "optOutUrl" : "url2",
                          "defaultState" : "optout"
                        }
                        ]
                      }
